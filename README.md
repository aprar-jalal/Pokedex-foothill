# Pokedex CLI

A command-line Pokedex application built with **TypeScript** that allows users to explore the Pokemon world, catch Pokemon, and manage their own Pokedex.

The application uses the **PokeAPI** to fetch real Pokemon data and provides an interactive REPL (Read-Eval-Print Loop) experience.

---

## Features

- Explore Pokemon world locations
- Navigate through location areas using pagination
- Explore specific areas and discover available Pokemon
- Catch Pokemon using a probability-based catching system
- Store caught Pokemon in your personal Pokedex
- Inspect caught Pokemon details
- Cache API responses to improve performance

---

## Technologies

- TypeScript
- Node.js
- PokeAPI
- Fetch API
- Vitest (Testing)
- Git
